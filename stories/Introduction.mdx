All components and design tokens are mapped to match the Figma specifications.

## ðŸŽ¯ Design Tokens

Our design system uses CSS variables that map directly to Figma variables, making it easy to maintain consistency between design and code.

### Color Tokens

- `--color-bg`: Main background color
- `--color-bg-subtle`: Subtle background for sections
- `--color-card`: Card background
- `--color-border`: Border color
- `--color-text`: Primary text color
- `--color-text-muted`: Secondary text color
- `--color-primary`: Primary brand color
- `--color-accent`: Accent color
- `--color-success`: Success state color
- `--color-warning`: Warning state color
- `--color-danger`: Danger/error state color
- `--color-info`: Info state color

### Radius Tokens

- `--radius-sm`: Small radius (0.375rem)
- `--radius-md`: Medium radius (0.5rem)
- `--radius-lg`: Large radius (0.75rem)

### Shadow Tokens

- `--shadow-surface`: Subtle shadow for elevated surfaces
- `--shadow-focus`: Focus ring shadow
